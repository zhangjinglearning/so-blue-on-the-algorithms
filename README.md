## percolation

在物理、化学和材料科学中，渗流（从拉丁语Percōlāre而来，意为“过滤”或“涓流”）指的是液体通过多孔材料时的运动和过滤行为。

例如：过滤咖啡：溶液为水，可渗透的物质是咖啡粉，溶质为带给咖啡色香味的可溶性化合物。

如果最下面一层被连通，就说明被渗透了。
